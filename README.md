# Google Search Scraper

A simple Python automation tool built with Selenium and BeautifulSoup that performs Google searches, extracts search result titles, and saves them into a text file.

---

## Features

- Automated Google search
- Extracts result titles (`h3` tags)
- Saves results to `names.txt`
- Custom Chrome options for smoother automation
- Easy to modify for different search categories

---

## Technologies Used

- Python
- Selenium
- BeautifulSoup4
